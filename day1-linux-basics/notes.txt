## Commands I learned today:
- whoami     → shows current user
- pwd        → shows current directory
- ls         → lists files
- ls -la     → lists files with permissions and hidden files
- mkdir      → creates a folder
- touch      → creates a file
- cat        → reads a file
- echo       → prints text or writes to file
- cd         → change directory
- hostname   → shows machine name
